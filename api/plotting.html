aplot(na1='', na2='', ninc='', degen='', scale='', vtk=None, quality=7, show_area_numbering=False, show_line_numbering=False, color_areas=False, show_lines=False, **kwargs)

APDL Command: APLOT

Displays the selected areas from na1 to na2 in steps of ninc.

Parameters

  • na1 (int, optional) â Minimum area to display.

  • na2 (int, optional) â Maximum area to display

  • ninc (int, optional) â Increment between minimum and maximum area.

  • degen â Degeneracy marker. This option is ignored when vtk=True.

  • str â Degeneracy marker. This option is ignored when vtk=True.

  • optional â Degeneracy marker. This option is ignored when vtk=True.

  • scale â Scale factor for the size of the degeneracy-marker star. The scale is the size in window space (-1 to 1 in both directions) (defaults to .075). This option is ignored when vtk=True.

  • vtk (bool, optional) â Plot the currently selected areas using pyvista. As this creates a temporary surface mesh, this may have a long execution time for large meshes.

  • quality (int, optional) â Quality of the mesh to display. Varies between 1 (worst) to 10 (best) when vtk=True.

  • show_area_numbering (bool, optional) â Display area numbers when vtk=True.

  • show_line_numbering (bool, optional) â Display line numbers when vtk=True.

  • color_areas (bool, optional) â Randomly color areas when True and vtk=True.

  • show_lines (bool, optional) â Plot lines and areas. Change the thickness of the lines with line_width=

Examples

Plot areas between 1 and 4 in increments of 2.

>>> mapdl.block(0, 1, 0, 1, 0, 1)
>>> mapdl.aplot(1, 4, 2)

Plot all areas and randomly color the areas. Label center of areas by their number.

>>> mapdl.aplot(show_area_numbering=True, color_areas=True)

kplot(np1='', np2='', ninc='', lab='', vtk=None, show_keypoint_numbering=True, **kwargs)

Displays the selected keypoints.

APDL Command: KPLOT

Parameters

  • np1 â Display keypoints from NP1 to NP2 (defaults to NP1) in steps of NINC (defaults to 1). If NP1 = ALL (default), NP2 and NINC are ignored and all selected keypoints [KSEL] are displayed.

  • np2 â Display keypoints from NP1 to NP2 (defaults to NP1) in steps of NINC (defaults to 1). If NP1 = ALL (default), NP2 and NINC are ignored and all selected keypoints [KSEL] are displayed.

  • ninc â Display keypoints from NP1 to NP2 (defaults to NP1) in steps of NINC (defaults to 1). If NP1 = ALL (default), NP2 and NINC are ignored and all selected keypoints [KSEL] are displayed.

  • lab â

    Determines what keypoints are plotted (one of the following):

    (blank) - Plots all keypoints.

    HPT - Plots only those keypoints that are hard points.

  • vtk (bool, optional) â Plot the currently selected lines using pyvista.

  • show_keypoint_numbering (bool, optional) â Display keypoint numbers when vtk=True.

Notes

This command is valid in any processor.

lplot(nl1='', nl2='', ninc='', vtk=None, show_line_numbering=True, show_keypoint_numbering=False, color_lines=False, **kwargs)

APDL Command: LPLOT

Displays the selected lines.

Parameters

  • nl1 â Display lines from NL1 to NL2 (defaults to NL1) in steps of NINC (defaults to 1). If NL1 = ALL (default), NL2 and NINC are ignored and display all selected lines [LSEL].

  • nl2 â Display lines from NL1 to NL2 (defaults to NL1) in steps of NINC (defaults to 1). If NL1 = ALL (default), NL2 and NINC are ignored and display all selected lines [LSEL].

  • ninc â Display lines from NL1 to NL2 (defaults to NL1) in steps of NINC (defaults to 1). If NL1 = ALL (default), NL2 and NINC are ignored and display all selected lines [LSEL].

  • vtk (bool, optional) â Plot the currently selected lines using pyvista.

  • show_numbering (bool, optional) â Display line and keypoint numbers when vtk=True.

  • show_keypoints (bool, optional) â Display keypoints associated with current lines. Only valid with vtk=True.

  • show_keypoint_numbering (bool, optional) â Number keypoints. Only valid when show_keypoints is True

  • **kwargs â See help(pyvista.plot) for more keyword arguments related to visualizing using vtk.

Examples

>>> mapdl.lplot(vtk=True, cpos='xy', line_width=10)

Notes

Mesh divisions on plotted lines are controlled by the LDIV option of the /PSYMB command.

This command is valid in any processor.

nplot(knum='', vtk=None, **kwargs)

APDL Command: NPLOT

Displays nodes.

Parameters

  • knum (bool, int, optional) â

    Node number key:

    • False : No node numbers on display (default).

    • True : Include node numbers on display.

  • vtk (bool, optional) â Plot the currently selected nodes using pyvista. Defaults to current use_vtk setting as set on the initialization of MAPDL.

Examples

Plot using VTK while showing labels and changing the background

>>> mapdl.prep7()
>>> mapdl.n(1, 0, 0, 0)
>>> mapdl.n(11, 10, 0, 0)
>>> mapdl.fill(1, 11, 9)
>>> mapdl.nplot(knum=True, vtk=True, background='w', color='k',
                show_bounds=True)

Plot without using VTK

>>> mapdl.prep7()
>>> mapdl.n(1, 0, 0, 0)
>>> mapdl.n(11, 10, 0, 0)
>>> mapdl.fill(1, 11, 9)
>>> mapdl.nplot(vtk=False)

Notes

Only selected nodes [NSEL] are displayed. Elements need not be defined.

Plot the selected volumes.

APDL Command: VPLOT

Parameters

  • nv1 â Display volumes from NV1 to NV2 (defaults to NV1) in steps of NINC (defaults to 1). If NV1 = ALL (default), NV2 and NINC are ignored and all selected volumes [VSEL] are displayed. Ignored when vtk=True.

  • nv2 â Display volumes from NV1 to NV2 (defaults to NV1) in steps of NINC (defaults to 1). If NV1 = ALL (default), NV2 and NINC are ignored and all selected volumes [VSEL] are displayed. Ignored when vtk=True.

  • ninc â Display volumes from NV1 to NV2 (defaults to NV1) in steps of NINC (defaults to 1). If NV1 = ALL (default), NV2 and NINC are ignored and all selected volumes [VSEL] are displayed. Ignored when vtk=True.

  • degen â Degeneracy marker. "blank" No degeneracy marker is used (default), or "DEGE". A red star is placed on keypoints at degeneracies (see the Modeling and Meshing Guide). Not available if /FACET,WIRE is set. Ignored when vtk=True.

  • scale â Scale factor for the size of the degeneracy-marker star. The scale is the size in window space (-1 to 1 in both directions) (defaults to .075). Ignored when vtk=True.

  • vtk (bool, optional) â Plot the currently selected volumes using pyvista. As this creates a temporary surface mesh, this may have a long execution time for large meshes.

  • quality (int, optional) â quality of the mesh to display. Varies between 1 (worst) to 10 (best). Applicable when vtk=True.

  • show_numbering (bool, optional) â Display line and keypoint numbers when vtk=True.

Examples

Plot while displaying area numbers

>>> mapdl.vplot(show_area_numbering=True)
