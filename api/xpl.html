ansXpl Class¶

MAPDL database explorer class description.

class ansys.mapdl.core.xpl.ansXpl(mapdl)¶

ANSYS database explorer class.

Examples

>>> from ansys.mapdl.core import launch_mapdl
>>> mapdl = launch_mapdl()
>>> xpl = mapdl.xpl

Returns

Return type

str

Examples

>>> xpl.close()
copy(newfile, option='')¶

Copy the current opened as a new file.

Parameters

  • newfile (str) – Name of the new file to create

  • option (str) – Option.

Examples

    >>      Remove existing output file tmpfile.full
goto(path)¶

Parameters

Examples
